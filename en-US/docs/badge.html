Badge 消息气泡

用于展示新消息的提示、数量

何时使用

  • 有新消息提示时
  • 常见于图标、文字标题、按钮等位置

普通类型

8
import React from 'react'
import Button from '@hi-ui/hiui/es/button'
import Badge from '@hi-ui/hiui/es/badge'
class Demo extends React.Component {
  render () {
    return (
      <Badge value={8}>
        <Button type='default'>最新报表</Button>
      </Badge>
    )
  }
}

限制最大显示值

44+
import React from 'react'
import Button from '@hi-ui/hiui/es/button'
import Badge from '@hi-ui/hiui/es/badge'
class Demo extends React.Component {
  render () {
    return (
      <Badge value={88} max={44}>
        <Button type='default'>最新报表</Button>
      </Badge>
    )
  }
}

隐藏气泡

90
设置 hidden 属性隐藏气泡
import React from 'react'
import Button from '@hi-ui/hiui/es/button'
import Badge from '@hi-ui/hiui/es/badge'
class Demo extends React.Component {
  render () {
    return (
      <Badge value={90} hidden>
        <Button type='default'>最新报表</Button>
      </Badge>
    )
  }
}

小红点

最新报表
小红点将优先于其他显示
import React from 'react'
import Button from '@hi-ui/hiui/es/button'
import Badge from '@hi-ui/hiui/es/badge'
class Demo extends React.Component {
  render () {
    return (
      <div>
        <Badge dot value='88' style={{marginRight: '32px'}}>最新报表</Badge>
        <Badge dot>
          <Button type='default'>最新报表</Button>
        </Badge>
      </div>
    )
  }
}

自定义内容

hot
小红点将优先于其他显示
import React from 'react'
import Button from '@hi-ui/hiui/es/button'
import Badge from '@hi-ui/hiui/es/badge'
class Demo extends React.Component {
  render () {
    return (
      <Badge value="hot">
        <Button type='default'>最新报表</Button>
      </Badge>
    )
  }
}

Badge Attributes

参数说明类型默认值
value气泡内容String | Number''
dot气泡是否显示为红点Booleanfalse
max气泡中显示数字的最大值Number99
hidden是否隐藏气泡Booleanfalse
以下调整将随 2.0.0 版本发布，敬请期待
