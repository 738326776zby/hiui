从 1.x 升级至 2.0

从 1.5.x 升级

为统一 API 规范，以下组件 API 在 2.0.0 中做了修正

基于 1.5.x 及之前版本开发的模板，可通过为组件批量增加 legacy={true} 标识，实现保持原有功能的目的（无法混用新旧版 API）。

  • Checkbox
  • Counter
  • DatePicker
  • Input
  • Radio
  • Select
  • Tabs
  • Tree
  • Upload

以 Checkbox 组件示例： 在 src 目录批量搜索 <Checkbox 替换为 <Checkbox legacy={true} 即可。
