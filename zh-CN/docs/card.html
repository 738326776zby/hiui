Card 卡片

用来展示不同类型信息或信息组的轻量容器

何时使用

  • 当一组同类信息不能自然形成区块感时，需借助卡片来组织
  • 当一个应用或功能由大量信息组成，需要通过重点信息快速识别时
  • 后台展示概览信息时

内容

卡片可以展示多种类型的信息：

  • 文本、数字
  • 段落
  • 信息图表，如环形图、柱状图、表格、折线图等
  • 图片、图片组
  • 图标
  • 列表

import React from 'react'
import Grid from '@hi-ui/hiui/es/grid'
import Card from '@hi-ui/hiui/es/card'
class Demo extends React.Component {
  render() {
    const {Row, Col} = Grid
    const colors = ['#46bc99', '#37d5fa', '#b450de', '#fadb14']
    const cols = colors.map((color, index) => {
      return (
        <Col span={6} key={index}>
          <Card hoverable style={{borderLeft: `2px solid ${color}`,}}>
            <p>无标题</p>
            <p>鼠标移入悬浮效果</p>
            <p>其它额外内容</p>
          </Card>
        </Col>
      )
    })
    return (
      <React.Fragment>
        <Row gutter={true}>
          {cols}
        </Row>
      </React.Fragment>
    )
  }
}
