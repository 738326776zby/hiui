# HiBaseTable

为满足开发对 table 的多样化需求，特基于[BaseTable](https://autodesk.github.io/react-base-table/docs/get-started)进行封装

## 以下情况下建议使用HIBaseTable

 - 数据量较多时候
 - table 自定义度较高时

## 使用方法


1. 安装基础依赖 react-base-table 

```shell
# npm
npm install react-base-table --save
# yarn
yarn add react-base-table

```
