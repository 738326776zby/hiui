# CodeEditor 代码编辑器

一种可内嵌于在页面中，进行相关代码编辑

- 基于开源库[codemirror](https://codemirror.net/index.html#description) 写的一些示例

## 使用方法

1. 安装 codemirror && react-codemirror2

```shell
npm install react-codemirror2 codemirror --save
```
