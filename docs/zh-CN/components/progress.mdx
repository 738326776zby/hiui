## Progress Attributes

| 参数      | 说明                                         | 类型                | 可选值                                         | 默认值    |
| --------- | -------------------------------------------- | ------------------- | ---------------------------------------------- | --------- |
| apperance | 进度条形状                                   | string              | 'bar' \| 'circle' \| 'dashboard'               | 'bar'     |
| size      | 进度条大小                                   | string              | 'large' \| 'middle' \| 'small'                 | -         |
| content   | 设置显示文案的内容，不设为当前进度百分比     | string \| ReactNode | -                                              | -         |
| showInfo  | 是否显示文本                                 | boolean             | true \| false                                  | false     |
| type      | 进度条类型                                   | string              | 'primary' \| 'success' \| 'warning' \| 'error' | 'primary' |
| radius    | 进度条半径，仅在 `apperance='circle'` 时有效 | number              | -                                              | 40        |
| placement | 文字显示位置，需配合 height 使用             | string              | 'inside' \| 'outside'                          | 'outside' |
