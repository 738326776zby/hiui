# Stepper 步骤

用来标识和组织完成一个任务时的信息

## 何时使用

- 当完成的目标需要分布来执行，且带有明确的先后次序
- 一个任务的每一步都需要较为复杂的信息和执行操作

## 基础用法

import DemoBase from '../../demo/stepper/section-base.jsx'

<DemoBase />

## 图标用法

stepper 图标用法传入 src 值

import DemoIcon from '../../demo/stepper/section-icon.jsx'

<DemoIcon />

## 上下结构

stepper 上下结构只需加入 up=true；只有上下结构有 text 描述

import DemoUpDown from '../../demo/stepper/section-up-down.jsx'

<DemoUpDown />

## 上下结构的图标用法

stepper 上下结构的图标用法需要加入 up=true，并在数值列表中添加 src 值

import DemoUpDownIcon from '../../demo/stepper/section-up-down-icon.jsx'

<DemoUpDownIcon />

## 竖直方向

stepper 竖直方向只需传入属性 vertical=true

## 竖直方向图标用法

stepper 竖直方向只需传入属性 vertical=true

## Props

| 参数       | 说明                            | 类型       | 可选值                     | 默认值       |
| ---------- | ------------------------------- | ---------- | -------------------------- | ------------ |
| className  | 样式名                          | string     | -                          | -            |
| data       | 步骤数据项                      | DataItem[] | -                          | -            |
| current    | 当前步骤位置索引，从 0 开始计数 | number     | -                          | -            |
| placement  | 位置                            | string     | 'vertical' \| 'horizontal' | 'horizontal' |
| itemLayout | 步骤点的布局方式                | string     | 'vertical' \| 'horizontal' | 'vertical'   |

## Type

**_DateItem_**

| 参数    | 说明        | 类型                | 可选值 | 默认值 |
| ------- | ----------- | ------------------- | ------ | ------ |
| title   | 显示文案    | string \| ReactNode | -      | -      |
| content | 内容文案    | string \| ReactNode | -      | -      |
| icon    | 自定义 icon | string \| ReactNode | -      | -      |
