## Props

| 参数        | 说明               | 类型                | 可选值                                                                 | 默认值 |
| ----------- | ------------------ | ------------------- | ---------------------------------------------------------------------- | ------ |
| type        | 设置输入框类型     | string              | 'text' \| 'textarea' \| 'id' \| 'tel' \| 'card' \| 'amount' \| 'email' | 'text' |
| disabled    | 是否禁用           | boolean             | true \| false                                                          | false  |
| required    | 是否必填           | string \| ReactNode | true \| false                                                          | false  |
| prefix      | 前缀               | string \| ReactNode | -                                                                      | -      |
| suffix      | 后缀               | string \| ReactNode | -                                                                      | -      |
| placeholder | 占位符             | string \| ReactNode | -                                                                      | -      |
| onFocus     | 获得焦点时触发     | (e: Event) => void  | -                                                                      | -      |
| onBlur      | 失去焦点时触发     | (e: Event) => void  | -                                                                      | -      |
| onKeyDown   | 触发 keydown 事件  | (e: Event) => void  | -                                                                      | -      |
| onKeyPress  | 触发 keypress 事件 | (e: Event) => void  | -                                                                      | -      |
| onInput     | 触发 input 事件    | (e: Event) => void  | -                                                                      | -      |
| onChange    | 值改变时触发       | (e: Event) => void  | -                                                                      | -      |
| onKeyUp     | 触发 keyup 事件    | (e: Event) => void  | -                                                                      | -      |
