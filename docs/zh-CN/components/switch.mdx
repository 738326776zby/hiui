## Props

| 参数     | 说明                     | 类型                     | 可选值        | 默认值  |
| -------- | ------------------------ | ------------------------ | ------------- | ------- |
| content  | 开关状态文案展示         | string[] \| ReactNode [] | -             | -       |
| checked  | 指定开关选中状态         | boolean                  | true \| false | -       |
| disabled | 是否禁用                 | boolean                  | true \| false | -       |
| size     | 开关尺寸                 | string                   | large         | default |
| onChange | 开关选中状态改变时的回调 | （checked）=>void        | large         | default |
