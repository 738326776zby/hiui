# Switch 开关

一种通过开启和关闭来控制两种状态的控制器组件

## 何时使用

- 有两种状态，通常切换时会引起状态的改变
- 常用于设置参数、控制面板等场景

## 基础用法

import DemoBase from '../../demo/switch/section-base.jsx'

<DemoBase />

## Props

| 参数     | 说明                     | 类型                    | 可选值               | 默认值    |
| -------- | ------------------------ | ----------------------- | -------------------- | --------- |
| content  | 开关状态文案展示         | string[] \| ReactNode[] | -                    | -         |
| checked  | 指定开关选中状态         | boolean                 | true \| false        | -         |
| disabled | 是否禁用                 | boolean                 | true \| false        | false     |
| size     | 开关尺寸                 | string                  | 'default' \| 'large' | 'default' |
| onChange | 开关选中状态改变时的回调 | （checked）=>void       | -                    | -         |
