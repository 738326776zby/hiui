# CodeEditor 代码编辑器

一种可内嵌于在页面中，进行相关代码编辑

- 基于开源库[codemirror](https://codemirror.net/index.html#description)做的二次封装
- 完全具备开源库 codemirror 的全功能，请放心大胆的使用

## CodeEditor

> 详细信息请参考 [CodeMirror](https://codemirror.net/doc/manual.html#config)

### props

| 参数         | 说明                                                 | 类型    | 可选值        | 默认值 |
| ------------ | ---------------------------------------------------- | ------- | ------------- | ------ |
| showTooltip  | 是否展示**文字提醒**                                 | boolean | true \| false | true   |
| tooltipTitle | 文字提醒内容                                         | string  | -             | -      |
| toolbarsName | 对应[Quill](http://doc.quilljs.cn/1409378)工具栏名称 | string  | -             | -      |
