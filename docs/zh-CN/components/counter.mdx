## Counter Attributes

| 参数      | 说明         | 类型             | 可选值 | 默认值 |
| --------- | ------------ | ---------------- | ------ | ------ |
| value     | 默认值       | String \| Number | -      |
| step      | 每次改变步数 | Number           | -      |
| min       | 最小值       | Number           | -      |
| max       | 最大值       | Number           | -      |
| disabled  | 是否禁用     | Boolean          | -      |
| className | String       | -                | -      |

## Counter Events

| 参数     | 说明             | 回调参数                      |
| -------- | ---------------- | ----------------------------- |
| onChange | 点击加减号时触发 | (event: Event, value: 原始值) |
