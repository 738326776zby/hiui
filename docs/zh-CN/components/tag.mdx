# Tag 标签

用来标记信息的属性，用以区分信息

## 何时使用

有新消息提示时

常见于图标、文本标题、按钮等位置

## Props

### Tag

| 参数       | 说明             | 类型                    | 可选值                                          | 默认值    |
| ---------- | ---------------- | ----------------------- | ----------------------------------------------- | --------- |
| type       | 设置状态         | string                  | 'primary' \| 'success' \| 'warning' \| 'danger' | 'primary' 
| appearance | 设置样式类型     | string                  | 'default' \| 'line'                             | 'default' |
| shape      | 设置形状     | string                  | 'round' \| 'square'                             | 'round' |
| color      | 标签色      | string                  |  -                          | - |
| onClick    | 点击按钮时的回调 | (e: MouseEvent) => void | -                          | -         |


### Tag.Group

| 参数       | 说明             | 类型                    | 可选值                                          | 默认值    |
| ---------- | ---------------- | ----------------------- | ----------------------------------------------- | --------- |
| data   | 展示数据      | TagNode[]                 | -                 | - |
| editable   | 标签组是否可以修改、新增、删除    | boolean                  | true \| false                        | false |
| onAdd      | 标签新增后触发      | (addNode:TagNode, index:number) => void                  | -                 | - |
| onEdit     | 标签修改后触发      | (editNode:TagNode, index:number) => void                  | -                 | - |
| onDelete   | 标签删除后触发      | (deleteNode:TagNode, index:number) => void                  | -                 | - |

## Type

### TagNode

| 参数              | 说明                                                       | 类型                             | 可选值          | 默认值 |
| ----------------- | ---------------------------------------------------------- | -------------------------------- | --------------- | ------ |
| title            | 标题                                             | string                           | - | -
| tagId            | 标签唯一 id                                            | string \| number                           | - | -
| closable | 标签是否可以关闭     | boolean                  | true \| false                        | false |
| editable | 标签是否可以修改    | boolean                  | true \| false                        | false |
