# Card 卡片

用来展示不同类型信息或信息组的轻量容器

## 何时使用

当一组同类信息不能自然形成区块感时，需借助卡片来组织

当一个应用或功能由大量信息组成，需要通过重点信息快速识别时

后台展示概览信息时

## 内容

卡片可以展示多种类型的信息：

- 文本、数字
- 段落
- 信息图表，如环形图、柱状图、表格、折线图等
- 图片、图片组
- 图标
- 列表

## Props

| 参数      | 说明                       | 类型                | 可选值                          | 默认值    |
| --------- | -------------------------- | ------------------- | ------------------------------- | --------- |
| title     | 设置卡片标题               | string \| ReactNode | -                               | -         |
| size      | 设置卡片大小               | string              | 'small' \| 'default' \| 'large' | 'default' |
| type      | 设置卡片类型               | string              | 'simple' \| 'default'           | 'default' |
| hoverable | 鼠标移入卡片时是否显示阴影 | boolean             | true \| false                   | false     |
| extra     | 卡片右上角的扩展           | ReactNode           | -                               | -         |
| extraType | 卡片右上角的扩展显示模式   | string              | 'default' \| 'hover'            | 'default' |
| coverUrl  | 卡片的封面图片链接         | string              | -                               | -         |
| content   | 卡片的描述信息             | string \| ReactNode | -                               | -         |
