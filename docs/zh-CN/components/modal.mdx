# Modal

模态对话框

## 基础用法

import DemoBase from '../../demo/modal/section-base.jsx'

<DemoBase />

## 自定义尺寸

- 通过 size 自定义尺寸，可使用 `large`、`normal`、`small`，默认为 `normal`

import DemoSize from '../../demo/modal/section-size.jsx'

<DemoSize />

## 自定义按钮

- 通过 footers 自定义底部的按钮，并执行对应自定义事件
- 通过 closeBtn 取消右上角关闭按钮

import DemoBtn from '../../demo/modal/section-btn.jsx'

<DemoBtn />

## 嵌套用法

- 未传入 title 及 closeBtn 为 false，可取消 title 部分
- footers 传入空数组，可取消底部按钮

import DemoNest from '../../demo/modal/section-nest.jsx'

<DemoNest />

## Props

| 参数          | 说明                       | 类型                       | 可选值        | 默认值  |
| ------------- | -------------------------- | -------------------------- | ------------- | ------- |
| title         | 标题                       | string \| ReactNode        | -             | -       |
| visible       | 是否显示对话框             | boolean                    | true \| false | false   |
| closeable     | 是否展示右上角关闭按钮     | boolean                    | true\|false   | false   |
| maskCloseable | 是否允许点击蒙层关闭对话框 | boolean                    | true\|false   | false   |
| cancelText    | 取消按钮文案               | string                     | -             | '取消'  |
| onCancel      | 取消事件触发时的回调       | (e: MouseEvent) => void    | -             | -       |
| confirmText   | 确认按钮文案               | string                     | -             | '确定'  |
| onConfirm     | 确认事件触发时的回调       | (e: MouseEvent) => void    | -             | -       |
| size          | 对话框尺寸                 | string                     | large         | default |
| style         | 自定义对话框样式           | object                     | -             | -       |
| footer        | 自定义气对话框底部         | string \|ReactNode \| null | -             | -       |

## Methods

`Modal.open({title, closeable, size, footer，style，onConfirm，confirmText，onCancel，maskCloseable，cancelText})`

| 参数          | 说明                       | 类型                       | 可选值      | 默认值  |
| ------------- | -------------------------- | -------------------------- | ----------- | ------- |
| title         | 标题                       | string \| ReactNode        | -           | -       |
| closeable     | 是否展示右上角关闭按钮     | boolean                    | true\|false | false   |
| maskCloseable | 是否允许点击蒙层关闭对话框 | boolean                    | true\|false | false   |
| cancelText    | 取消按钮文案               | string                     | -           | '取消'  |
| onCancel      | 取消事件触发时的回调       | (e: MouseEvent) => void    | -           | -       |
| confirmText   | 确认按钮文案               | string                     | -           | '确定'  |
| onConfirm     | 确认事件触发时的回调       | (e: MouseEvent) => void    | -           | -       |
| size          | 对话框尺寸                 | string                     | large       | default |
| style         | 自定义对话框样式           | object                     | -           | -       |
| footer        | 自定义气对话框底部         | string \|ReactNode \| null | -           | -       |
