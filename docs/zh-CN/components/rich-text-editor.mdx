# 富文本编辑器

一种可内嵌于浏览器，所见即所得的文本编辑器

- 基于开源库[quill](http://doc.quilljs.cn/)做的二次封装
- 完全具备开源库quill的全功能，请放心大胆的使用

## 基础用法

import DemoBase from '../../demo/rich-text-editor/section-base.jsx'

<DemoBase />

## 工具栏提醒功能

import DemoTooltip from '../../demo/rich-text-editor/section-tooltip.jsx'

<DemoTooltip />

## RichTextEditor

### Props

>  详细信息请参考 [react-quill](https://github.com/zenoamaro/react-quill)


| 参数  | 说明                                   | 类型   | 可选值 | 默认值             |
| ----- | -------------------------------------- | ------ | ------ | ------------------ |
| theme | 根据 HiUI 相关规范，进行了定制**主题** | string | 'snow' \| 'bubble' | - |

## QuillBarTooltip

### props

| 参数        | 说明                 | 类型    | 可选值        | 默认值 |
| ----------- | -------------------- | ------- | ------------- | ------ |
| showTooltip | 是否展示**文字提醒** | boolean | true \| false | true   |
| tooltipTitle |  文字提醒内容 | string | - | -   |
| toolbarsName |  对应[Quill](http://doc.quilljs.cn/1409378)工具栏名称 | string | - | -   |
