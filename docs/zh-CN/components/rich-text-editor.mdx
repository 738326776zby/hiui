# 富文本编辑器

一种可内嵌于浏览器，所见即所得的文本编辑器

- 基于开源组件[quill](http://doc.quilljs.cn/)做的二次封装

## RichTextEditor

### Props

| 参数  | 说明                                   | 类型   | 可选值 | 默认值             |
| ----- | -------------------------------------- | ------ | ------ | ------------------ |
| theme | 根据 HiUI 相关规范，进行了定制**主题** | string |        | 'snow' \| 'bubble' | - |

## QuillBarTooltip

### props

| 参数        | 说明                 | 类型    | 可选值        | 默认值 |
| ----------- | -------------------- | ------- | ------------- | ------ |
| showTooltip | 是否展示**文字提醒** | boolean | true \| false | true   |
