## Props

**_Form_**

| 参数           | 说明                                                                                  | 类型    | 可选值                     | 默认值  |
| -------------- | ------------------------------------------------------------------------------------- | ------- | -------------------------- | ------- |
| model          | 表单数据                                                                              | object  | -                          | -       |
| rules          | 表单验证规则，用法参考 [async-validator](https://github.com/yiminghe/async-validator) | object  | -                          | -       |
| labelWidth     | label 宽度，可用任意 CSS 长度单位                                                     | string  |                            |
| labelPlacement | label 放置的位置                                                                      | string  | 'right' \| 'left' \| 'top' | 'right' |
| placement      | 是否横向排列                                                                          | boolean | true \| false              | false   |

**_Form.Item_**

| 参数     | 说明                       | 类型    | 可选值        | 默认值 |
| -------- | -------------------------- | ------- | ------------- | ------ |
| field    | 表单域对应 model 的字段    | string  | -             | -      |
| label    | 标签文本，默认自动添加冒号 | string  | -             | -      |
| required | 是否必填                   | boolean | true \| false | false  |

## Methods

| 方法名                        | 说明               |
| ----------------------------- | ------------------ |
| validate(callback)            | 对整个表单进行校验 |
| validateField(prop, callback) | 对表单字段进行校验 |
| resetValidates()              | 重置整个表单的验证 |
