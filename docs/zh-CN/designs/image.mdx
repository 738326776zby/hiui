# 图片

图片在不同使用环境下，有不同的使用规范。卡片圆角统一为：2px

## 通栏图片

一般应用在首页 Banner、背景图、运营宣传等场景，图片比例 3:1。

## 预览图片

应用于预览图片的场景，点击放大缩略图后的图片。灰色部分为浏览器区域，蓝色为最大展示区。红色区域是距离浏览器边缘距离。 上下间距 48 px，左右间距 96 px。

- 当图片超出高度时，高度缩小为最大值，宽度等比例缩小。
- 当图片超出宽度时，宽度缩小为最大值，高度等比例缩小。
- 如果图片宽和高都超出，先等比缩小宽度；如果高度依旧超出，再等比缩小图片高度。

图片根据浏览器自适应宽高。

## 缩略图

一般用户列表、上传图片、Table 组件的单元格中，以缩略图的形式存在。图片比例 1:1 。
