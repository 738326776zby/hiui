## Progress Attributes

| 参数      | 说明                                   | 类型                          | 可选值                              | 默认值         |
| --------- | -------------------------------------- | ----------------------------- | ----------------------------------- | -------------- |
| apperance | 进度条类型                             | string                        | bar \| circle \| dashboard          | bar            |
| size      | 进度条大小                             | string                        | large \| middle \| small            | -              |
| content   | 显示文本                               | string \| string \| ReactNode | -                                   | 当前进度百分比 |
| showInfo  | 是否显示文本                           | boolean                       | true \| false                       | false          |
| type      | 进度条类型                             | string                        | primary \| success \| warn \| error | primary        |
| radius    | 环形进度条半径                         | number                        | -                                   | 40             |
| placement | 文字在进度条内显示，需配合 height 使用 | string                        | inside \| outside                   | outside        |
