# 更新日志

`HIUI` 遵循 [Semantic Versioning 2.0.0](http://semver.org/lang/zh-CN/) 语义化版本规范。

> 版本格式：主版本号.次版本号.修订号，版本号递增规则如下：
> 1. 主版本号：当你做了不兼容的 API 修改，
> 1. 次版本号：当你做了向下兼容的功能性新增，
> 1. 修订号：当你做了向下兼容的问题修正。
> 先行版本号及版本编译信息可以加到“主版本号.次版本号.修订号”的后面，作为延伸。
