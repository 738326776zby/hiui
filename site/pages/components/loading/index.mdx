## Props

| 参数     | 说明                             | 类型                | 可选值                    | 默认值  |
| -------- | -------------------------------- | ------------------- | ------------------------- | ------- |
| size     | 组件大小                         | string              | large \| default \| small | default |
| content  | 自定义加载中状态文案             | string \| ReactNode | -                         | -       |
| visible  | 是否显示加载动画                 | boolean             | true \| false             | true    |
| duration | 持续时间，防止闪烁（单位：毫秒） | number              | -                         | -       |

## Methods

`Loading.open({target, content, size, duration})`

| 参数     | 说明                             | 类型                | 可选值                    | 默认值  |
| -------- | -------------------------------- | ------------------- | ------------------------- | ------- |
| target   | 需要`Loading` 遮罩的元素         | ReactNode           | -                         | -       |
| content  | 自定义描述加载中文案             | string \| ReactNode | -                         | -       |
| duration | 持续时间，防止闪烁（单位：毫秒） | number              | -                         | -       |
| size     | 组件大小                         | string              | large \| default \| small | default |
