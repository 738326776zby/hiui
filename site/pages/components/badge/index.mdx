常用的消息气泡提示组件


## 普通类型

import DemoNormal from './section-normal.jsx'

<DemoNormal/>

## 限制最大显示值

<DemoMax/>


## 隐藏气泡

import DemoHidden from './section-hidden.jsx'

<DemoHidden/>


## 小红点

小红点将优先于其他显示

import DemoDot from './section-dot.jsx'

<DemoDot/>


## 自定义内容

可以显示 String 类型的内容

import DemoCustom from './section-custom.jsx'

<DemoCustom/>


## Badge Attributes

| 参数 | 说明 | 类型 | 默认值 |
| -------- | ----- | ---- | ---- |
| value | 气泡内容 | string \| number | '' |
| dot | 气泡是否显示为红点 | boolean | false | 
| max | 气泡中显示数字的最大值 |  number  | 99 |
| hidden | 是否隐藏气泡 | boolean | false |
